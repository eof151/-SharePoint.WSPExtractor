Intro

I want to publish my new windows form app on CodePlex Gallery for SharePoint 2010/2013.

This tool called "SharePoint.WSPExtractor" will extract the installed solutions as wsp-files from SharePoint Farm to a specified folder.

At the “source code” section I put two versions: 2010 and 2013

Usage

You must start this application only on SharePoint server machine.

 

Requirements:

- .NET 3.5/4.0

- Microsoft.SharePoint.dll
